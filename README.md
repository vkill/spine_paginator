# SpinePaginator

Paginator for Spine. [DEMO]()


## Installation

### Rails

Add this line to your application's Gemfile:

    gem 'spine_paginator'

And then execute:

    $ bundle


## Usage

see https://github.com/vkill/spine_paginator/examples/spine_pagination.coffee
